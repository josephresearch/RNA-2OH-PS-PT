# Supplementary Data

All-atom and coarse-grained simulation data supporting the supplementary figures.

## Subdirectories

| Directory | Contents |
|-----------|----------|
| `CAGx6/MgCl2/backbone_radial_distributions/` | Mg2+ and water RDFs vs. backbone Op — rCAGx6, dCAGx6, rCmAGx6; 3 replicates × 2 temperatures |
| `CAGx6/MgCl2/sugar_radial_ditributions/` | Mg2+ RDFs vs. sugar O2'/H2' — same sequences and conditions |
| `CAGx6/MgCl2/persistance_lengths/` | Persistence lengths for rCAGx6, dCAGx6, rCmAGx6 |
| `CAGx6/CaCl2/backbone_radial_distributions/` | Ca2+ and water RDFs vs. backbone Op — same sequences and conditions, 150 mM CaCl2 |
| `CAGx10/radial_distributions/` | Mg2+ and water RDFs for d/r(CAG)10, averaged over replicates |
| `CAGx10/radius_of_gyration/` | Rg time series for d/r(CAG)10 |
| `CUGx6/radius_of_gyration/` | Rg time series for r(CUG)6 |
| `coarse-grained_order_parameters/` | Orderness S, shape anisotropy A, and interchain crosslinks N vs. stiffness κ |

See the `readme.md` in each subdirectory for file listings and data format details.
